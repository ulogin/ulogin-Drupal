Форма авторизации uLogin через социальные сети. Улучшенный аналог loginza.

== Description ==

uLogin — это инструмент, который позволяет пользователям получить единый доступ к различным Интернет-сервисам без необходимости повторной регистрации,
а владельцам сайтов — получить дополнительный приток клиентов из социальных сетей и популярных порталов (Google, Яндекс, Mail.ru, ВКонтакте, Facebook и др.)

== Installation ==

Распакуйте содержимое архива в каталог модулей, после чего активируйте модуль на странице администрирования модулей.

== How To Use ==

После установки на формах входа и регистрации будет добавлен виджет uLogin.
Вы можете создать свой виджет и редактировать его самостоятельно на сайте http://ulogin.ru/. 
Для создания виджета необходимо: 

1. Войти в Личный Кабинет (ЛК) на сайте http://ulogin.ru/
2. Добавить свой сайт к списку "Мои сайты"
3. На вкладке "Виджеты" добавить новый виджет
4. Внимание: Для работы модуля uLogin в ЛК необходимо включить в обязательных полях профиля поле Еmail
5. В настройках модуля uLogin необходимо указать ID Вашего виджета в соответствующем поле
